# QuestHive

A decentralized gamified lifestyle app for tracking habits and daily goals built on Stacks blockchain.

## Features
- Create and manage quests (daily goals/habits)
- Complete quests to earn rewards
- Level up system based on quest completion
- Achievement badges stored as NFTs
- In-app currency (Honey tokens) for rewards

## Smart Contracts
- `quests.clar`: Core quest management and completion logic
- `honey-token.clar`: Fungible token for in-app rewards 
- `badges.clar`: NFT contract for achievement badges
- `player.clar`: Player profile and stats management
